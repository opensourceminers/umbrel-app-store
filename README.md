# ⚡ Open Source Miners — Umbrel Community App Store

A community app store for [umbrelOS](https://umbrel.com).

## Add this store to your Umbrel

1. Open the **App Store** in umbrelOS
2. Scroll down → **Community App Stores**
3. Paste this repository's URL and click **Add**
4. Install the apps below

## Apps

### LightningMate

Profit-aware management & autopilot for your Lightning node — fee and rebalance
automation, channel suggestions with one-click open, a Forwards report, and a
Profit & Loss overview. Read-only by default; writes are opt-in.

Source: https://github.com/opensourceminers/lightningmate
